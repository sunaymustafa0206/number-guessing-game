# Number Guessing Game

A simple, interactive command-line game built with Python where the player tries to guess a randomly generated number.

## 🚀 Features
* Random number generation between 1 and 100.
* Input validation (prevents crashes on non-numeric input).
* Score tracking (counts the number of attempts).
* Play-again functionality.

## 🛠️ Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/number-guessing-game.git](https://github.com/your-username/number-guessing-game.git)

2.Navigate to the directory:

Bash
cd number-guessing-game
Run the game:

Bash
python main.py

📝 How to Play
The computer will pick a number between 1 and 100.

Enter your guess in the terminal.

The game will tell you if your guess is "Too high" or "Too low."

Keep guessing until you find the secret number!
